# binary-timer-8051
Бинарный секундомер для микроконтроллера 8051

# Команды для микроклавиатуры
- Клавиша 2 – прибавить 1
- Клавиша 1 – прибавить 0
- Клавиша 3 – старт
- Клавиша А – сброс

# Команды для клавиатуры ПК
- Num 1 – прибавить 1
- Num 2 – прибавить 0
- Enter – старт
- Backspace – сброс
